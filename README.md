## 项目结构

| 模块      | 功能            |
|:-------:|:-------------:|
| adapter | 导出的json配置格式   |
| common  | 多模组加载器之间的通用代码 |
| fabric  | fabric侧的专用代码  |
| forge   | forge侧的专用代码   |
